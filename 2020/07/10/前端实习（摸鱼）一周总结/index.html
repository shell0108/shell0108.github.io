1.公司简介

这家小公司就在学校附近，每天上下班都看得到学校，熬过去这两月又要回校读研了。老板是你电硕士毕业，他还喊我师弟，面试给放了不少水🐶

老板接外包，私企国企政府的都接。同事目前做国家电网的项目比较多、还有蜀能、智能环卫撒子的。有的像是门户网站，你会知道为什么我们的学校的选课系统、信息门户都是那样“简洁大方”，把页面逻辑跑通，能用就行，都是十几年前的web技术了。

技术氛围挺好，大家都是实习生，完成放飞自我，不懂就互相讨论，学得还是挺快的，反正注重结果，项目交付时候的时候一定要有自信。

没有产品经理、项目经理、测试人员这么一说，前后端直接面对客户开撕需求，真刺激

2.vscode调教

使用到git进行多人协作开发时候，我习惯使用

  • Git blame
  • Annotator
  • GitLens

这些插件，可以看到目前项目中代码每一行是谁什么时候写的，方便甩锅（逃

当时上家公司时候用它的目的是，有些代码写得我看不懂，知道是谁写的，方便找前辈问。最后偶尔看到自己刚入职时候的写的代码都想删了重写（哭）

还有css取色器插件，可以很方便的修改颜色

markdown preview enhanced 可以在vscode中渲染查看markdown文件，现在写博客都是使用的markdown格式，这样边写边看效果就会比较方便

vscode主题Monokai很经典好用，我sublime也是使用的这个主题配色

关于这些插件都可以写到.vscode/extensions.json 中，发给同事这个文件，vscode就会自动提示安装

增加按月显示考勤记录

这边使用到element-ui的时间控件，选择到月。一开始设置好绑定日期到格式、默认给我绑定为一个对象了…应该让它绑定为字符串，显示为“2020-05-01 00:00:00”这样， 后端api传参有三个，一个是被考核人员id，另外两个是考核的时间段，即起始时间和截止时间。现在考核的起始时间有了，那么考核截止时间应该为“2020-06-01 00:00:00”这样，于是要用到字符串剪切获取到当前月份，再将其转化为数字+1作为下个月，还要考虑到12月，年份要+1，月份变为1。于是代码会变成下面这样，这是对传入后端api字段的清洗工作

searchByMonth(row){
        let startTime = this.revocatRecordFilter.month;
        let endTime = ''
        if(startTime != null){
          let month = startTime.substring(5,7);
          let nextMonth = (parseInt(month)+1).toString();
          let year = parseInt(startTime.substring(0,4));
          endTime = startTime.substring(0,5)+nextMonth+startTime.substring(7)
          if(month == 12){
            nextMonth = "01";

          };
