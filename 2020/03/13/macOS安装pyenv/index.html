pyenv install 要安装python版本

这一步实在是太慢了，python官网被墙我也是没办法

1.于是转战淘宝源，下载好要安装的python版本

2.再手动创建～/.pyenv/cache目录,将下载好的python版本copy过来

3.又build fail了，按照提示用homebrew安装readline

4.可是homebrew又好慢啊

5.依然build failed, 查看报错信息丢去百度，原来是mac更新catalina后出现的问题

6.至此，安装新版本python完成，大部分时间都在等下载：）
